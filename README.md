# Distributed-power Open Standards Alliance

DOSA preserves open interface standards for board-mounted DC-DC power modules.

The archive focuses on public standards material that helps engineers compare, source, and reuse power module designs across compatible footprints and pinouts. It includes standards for Brick and Point-of-Load (POL) module ecosystems, with package outlines, PCB footprints, pin definitions, feature descriptions, and reference specification drawings.

## Purpose

DOSA exists to keep distributed-power module interfaces open, vendor neutral, and practical for engineering use.

Open standards reduce design risk when alternate sourcing, long product lifecycles, or footprint compatibility matter. The goal is not to promote a single manufacturer, but to preserve shared reference material that can be used by power electronics engineers, hardware designers, sourcing teams, and standards contributors.

## Standards Archive

The public website is available at [www.dosapower.org](https://www.dosapower.org/).

The current archive covers:

- Brick module standards
- Point-of-Load (POL) module standards
- Package and footprint reference material
- Public PDF specifications and engineering references
